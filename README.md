# ClickToCastButton

A World of Warcraft addon that adds a convenient button to the spellbook window for quick access to click-to-cast bindings.

## Features

- Adds a "Click-to-Cast" button to the bottom of the spellbook window
- One-click access to the click-to-cast binding interface
- Tooltip with description on hover

## Installation

1. Download the addon files
2. Extract to your World of Warcraft addons folder:
   - `World of Warcraft\_retail_\Interface\AddOns\ClickToCastButton`
3. Restart WoW or reload your UI with `/reload`

## Usage

1. Open your spellbook (default key: P)
2. Click the "Click-to-Cast" button at the bottom of the window
3. The click-to-cast binding interface will open

## Compatibility

- World of Warcraft: The War Within (11.x)
- Requires the modern spellbook (PlayerSpellsFrame)
